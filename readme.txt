#Telegram Bot – Voice Expense Tracker

This bot allows you to **record expenses via voice messages** in Telegram.  
It automatically transcribes speech, extracts structured expense data using AI, and saves it into **Google Sheets**.

---

## 📌 Features
- 🎙️ Voice input – send a voice message with your expense  
- 📝 Automatic transcription (AssemblyAI)  
- 🤖 AI-powered parsing of amount, date, and place (OpenAI)  
- 📊 Saves all expenses directly to Google Sheets  

---

## ⚙️ Requirements
- Python 3.9+
- Libraries: `python-telegram-bot`, `python-dotenv`, `assemblyai`, `openai`, `gspread`, `oauth2client`
